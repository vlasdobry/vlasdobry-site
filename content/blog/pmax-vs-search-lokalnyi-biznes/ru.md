# Performance Max vs Search: что выгоднее для локального бизнеса

Когда я взял в работу Google Ads для сервиса по ремонту бытовой техники в Bay Area, картина была такой: $500 потрачено, 1 лид. CPA — $500. Рыночный CPA для appliance repair в Bay Area — $50-90 за лид. Фактический CPA $307 — в 3-6 раз выше. CPA $500 в первую неделю — в 6-10 раз.

Через месяц CPA снизился до $13.91 за лид. Что произошло — разбираю по цифрам.

## Исходная ситуация: Search-only и три структурные ошибки

Клиент — KoxFix Solutions, ремонт холодильников, стиральных машин, плит и HVAC в 13 городах Bay Area. До моего прихода кампании были настроены так:

1. **Broad match без Smart Bidding.** 275 ключевых слов, 88% без единого показа. Система расходовала бюджет на мусорные запросы.
2. **Нет минус-слов.** Объявления показывались по запросам, не имеющим отношения к услуге.
3. **Quality Score 1-3** у 59% ключевых слов. Посадочные страницы — SPA без SSR, canonical всех страниц указывал на главную. Google не понимал, на какую страницу вести пользователя.

Результат: CPA $307 в среднем, 2 конверсии за 30 дней, алгоритм Smart Bidding в статусе «обучение» месяц.

## Что изменили: структура, а не бюджет

### Шаг 1. Реструктуризация Search

Вместо одной кампании с Broad match — шесть тематических кампаний по категориям техники. 55 ключевых слов вместо 275, все Exact match (на старте; позже расширили до 125 по мере добавления брендовых кампаний). Каждое слово ведёт на конкретную посадочную страницу.

Минус-слова: б/у, бесплатно, своими руками, DIY, инструкция, видео.

### Шаг 2. Запуск Performance Max

Для холодильников запустили PMax параллельно с Search. Бюджет: $20/день. Настроили фид, группу объектов, аудиторию.

**Результат первого дня PMax:**
- 2,625 показов
- 28 кликов
- 2 конверсии, ценность $225
- Потрачено $11.47
- ROAS 19.6x (реальный после корректировки Google — 6.5x)

### Шаг 3. Расширение PMax на другие категории

После успеха с холодильниками запустили PMax Luxury Refrigerator ($30/день) и PMax Washer/Dryer ($30/день).

## Что показали цифры: PMax vs Search по неделям

| Неделя | Показы | Конверсии | CPA |
|--------|--------|-----------|-----|
| W1 (13-19.04) | 277 | 1 | $270 |
| W2 (20-26.04) | 460 | 3 | $186 |
| W3 (27.04-03.05) | 5,311 | 7 | $31 |
| W4 (11-17.05) | — | 44 | $13.91 |

W3 — точка перелома. Именно тогда PMax вышел на рабочий режим. 5,311 показов, 7 конверсий, CPA упал с $186 до $31. На W4 PMax стал основным драйвером: 44 конверсии, CPA $13.91.

**Каналы внутри PMax:** 78% Display, 19% Cross-Network, 2% Search. То есть PMax находил лиды там, где Search-кампания их не видела.

Параллельно подтянулся Quality Score: с 2.43 до 3.72 (+53% за месяц). Impression Share вырос с 19.8% до 50.6%. Lost Rank упал с 80.2% до 30.1% — теперь основной ограничитель не качество, а бюджет.

## Когда Search всё ещё нужен

PMax не заменил Search полностью. Три сценария, где Search остаётся основным:

1. **Брендовые запросы.** Exact match на названия брендов техники (Samsung, LG, Whirlpool) дают самый горячий трафик. PMax здесь менее эффективен — он оптимизируется на конверсию, а не на брендовый охват.
2. **Узкие услуги.** «Ремонт льдогенератора Sub-Zero» — запрос с низкой частотностью, где PMax не соберёт достаточно данных для обучения. Search с Exact match отрабатывает точнее.
3. **Контроль над географией.** PMax может показываться за пределами заданного радиуса. Для бизнеса, работающего в 13 городах, Search даёт гарантию геотаргетинга.

## Выводы для локального бизнеса

1. **PMax + Search работают в паре.** PMax находит лиды через Display и Cross-Network, Search — через поисковый спрос. Одно без другого теряет охват.
2. **Exact match > Broad match для локального рынка.** Узкая семантика, небольшой бюджет — Broad match распыляет деньги. На старте лучше 55 точных ключей, чем 275 размытых.
3. **Качество посадочной страницы влияет на CPA напрямую.** SPA без SSR, canonical на главную — базовая техническая гигиена. Quality Score вырос на 53% после исправления.
4. **Не оценивай PMax по первому дню.** Google может отозвать конверсии в течение 48 часов. Реальный ROAS первого дня был 6.5x, а не 19.6x. Смотри на тренд по неделям.

AI Visibility Checker

Последнее обновление: 17 апреля 2026 г. (v2 — добавлены per-check числовые баллы)

Коротко: расширение не собирает персональные данные. Только анонимная статистика использования (сколько раз запустили, какой получили скор). Никакой контент страниц, URL, cookies, истории браузера не передаётся.

Кто мы

AI Visibility Checker — браузерное расширение для проверки готовности сайта к поиску в нейросетях (ChatGPT, Perplexity, Claude, YandexGPT, Gemini). Разработчик — Влас Фёдоров (vlasdobry@gmail.com).

Какие данные собираются

Расширение отправляет следующие данные в аналитику:

  • uid — случайный анонимный идентификатор (UUID v4), генерируется на устройстве, не связан с вашей личностью
  • event — тип события: установка, запуск аудита, экспорт отчёта, клик по ссылке
  • domain — только домен проверяемого сайта (например, example.com), без пути и параметров URL
  • score — итоговый скор аудита (0–100)
  • check_id — числовой балл каждой из 7 проверок (Schema.org, llms.txt, E-E-A-T, FAQ и т. д.). Это только цифры, без содержимого страницы
  • status_level — категория итогового скора (1–4)
  • locale — язык интерфейса браузера (например, ru или en)
  • version — версия расширения
  • ts — время события (Unix timestamp)

Агрегированные данные (например, «сколько сайтов не имеют llms.txt») могут использоваться для исследований рынка и публикаций на блоге, но без упоминания конкретных доменов.

Какие данные НЕ собираются

  • Имя, email, телефон — никакая персональная информация
  • IP-адрес не сохраняется в базе данных
  • Содержимое проверяемых страниц
  • Полные URL — только домен
  • Cookies, пароли, данные форм
  • История браузера
  • Данные из других вкладок, которые вы не проверяли через расширение

Зачем собираются данные

Статистика нужна только чтобы:

  • Понимать, сколько людей пользуются расширением
  • Улучшать продукт на основе реальных паттернов использования
  • Видеть средние скоры по аудитам (без привязки к конкретным пользователям)

Данные не продаются третьим лицам, не используются для рекламы, не передаются рекламным сетям.

Где хранятся данные

События сохраняются в базе данных Cloudflare D1 (SQLite) в регионе Восточной Европы. Доступ к базе имеет только разработчик.

Список ботов AI-поисковиков и настройки скоринга хранятся локально в chrome.storage.local на вашем устройстве и не синхронизируются с облаком.

Разрешения расширения

Расширение запрашивает следующие разрешения:

  • activeTab — доступ к активной вкладке, чтобы прочитать HTML для анализа (только при нажатии на иконку)
  • storage — сохранение настроек и анонимного идентификатора на вашем устройстве
  • scripting — инъекция скрипта анализа в активную вкладку (нужно после перезагрузки расширения)
  • alarms — периодическое обновление конфигурации (карты ботов, веса скоринга)

Расширение не работает в фоне, не читает данные с других вкладок автоматически, не следит за вашим браузингом.

Сторонние сервисы

  • Cloudflare — хостинг Worker и D1-базы (политика Cloudflare)
  • Google Chrome Web Store — доставка расширения (политика Google)

Других сторонних сервисов нет. Google Analytics, Yandex.Metrika, Facebook Pixel и подобные трекеры не используются.

Срок хранения

Данные аналитики хранятся до 24 месяцев, после чего удаляются автоматически. Если вы хотите удалить ваши данные раньше — напишите на vlasdobry@gmail.com, укажите анонимный uid (его можно найти в chrome://extensions → AI Visibility Checker → service worker → Application → Storage → userId). Удалим в течение 30 дней.

Ваши права

  • Удалить всё одним действием: удалите расширение через chrome://extensions. Локальные данные (userId) исчезнут автоматически.
  • Удалить историю событий с сервера: напишите нам — удалим по вашему uid.
  • Запросить копию данных: напишите — вышлем всё, что связано с вашим uid.
  • Отказаться от сбора: заблокируйте geo-checker-api.vlasdobry.workers.dev в файрволе или удалите расширение.

Изменения политики

При существенных изменениях этой политики мы покажем уведомление прямо в расширении через remote config. Дата последнего обновления указана в начале страницы.
